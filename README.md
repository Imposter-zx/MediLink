# MediLink

MediLink is a modern healthcare connectivity platform designed to bridge the gap between patients, pharmacies, and delivery services.

## Features

- **Patient Dashboard**: Manage health records and track medications.
- **Pharmacy Dashboard**: Process prescriptions and manage inventory.
- **Delivery Dashboard**: Track and manage medical deliveries.
- **Medication Management**: Centralized hub for medication tracking.

## Technology Stack

- **Frontend**: React.js with Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Imposter-zx/MediLink.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
